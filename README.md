# Customer Snapshot

Customer Snapshot is a simple customer lookup dashboard.

The user enters a customer's phone number and receives a summary showing:

- Customer name
- Total number of orders
- Lifetime purchase value
- Most recent purchase date
- Customer status

## Why I am building this

This project is based on a business system I previously designed to help employees understand the importance and purchase history of each customer before assisting them.

The original system helped improve customer service by making relevant customer information immediately visible.
